# Non-human Workflow

The non-host branch includes:
- rRNA removal with `ribodetector_cpu`
- Kraken2 taxonomic classification of rRNA reads
- collapse of non-rRNA reads into counted FASTA sequences
- tRNA-like assignment
- miRNA-like assignment
- Rfam class assignment
- export of remaining unknown sequences

Key outputs are written below `Analysis/<TYPE>/nonhuman/`.
